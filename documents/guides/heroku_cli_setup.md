## Heroku CLI Setup

### 1. Install Heroku CLI

Follow the [official instructions](https://devcenter.heroku.com/articles/heroku-cli).

### 2. Log in

Open your terminal and run:

```sh
heroku login
```

```sh
heroku --version
```
